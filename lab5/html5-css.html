HTML5 e CSS


HTML5

O HTML5 (Hyper Text Markup Language) é uma linguagem de markup utilizada para estruturar e apresentar conteúdo na Internet. É a quinta e última geração da linguagem, recomendada pelo Consórcio da World Wide Web.


Elementos HTML5
Elemento HTML5 Presença Descrição
head Cabeçalho de uma página Web. Presente em todas as páginas.
body Corpo de uma página Web. Presente em todas as páginas.
header Cabeçalho de uma página Web. Presente em todas as páginas.
main Conteúdo principal de uma página Web. Presente em todas as páginas.
footer Rodapé de uma página Web. Presente em todas as páginas.
h Headers. Presentes em todas as páginas.
p Parágrafo. Presente em todas as páginas.
br Quebra de linha. Presente em todas as páginas.
aside Divisor lateral. Não presente neste website.
details Define detalhes. Não presente neste website.
abbr Indica significado de uma abreviatura. Não presente neste website.


CSS

O CSS (Cascading Style Sheets) é uma linguagem de estilização utilizada para alterar a apresentação de documentos escritos em liniguagens de markup, tais como o HTML. É um dos elementos principais do desenvolvimento web, juntamente com HTML e JavaScript.


Elementos CSS
Elemento CSS Presença Descrição
* Seletor para todos os elementos de uam página. Presente.
body Seletor para o corpo da página. Presente.
header Seletor para o cabeçalho da página. Presente.
h Seletor para os títulos da página. Presente.
table Seletor para tabelas. Presente.
th Seletor para o cabeçalho de uma tabela. Presente.
td Seletor para os dados de uma tabela. Presente.
tr Seletor para uma linha de uma tabela. Presente.
:hover Seletor para quando um elemento é sobreposto pelo cursor. Presente.
:before Seletor utilizado para inserir algo antes do conteudo selecionado. Presente.
:after Seletor utilizado para inserir algo depois do conteudo selecionado. Não presente.
:active Seletor utilizado para identificar um elemento selecionado. Não presente.
